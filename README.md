# 💬 Discord Clone

Sistema de chat em tempo real inspirado no Discord, permitindo comunicação entre usuários por servidores e canais.

---

## 🎯 Objetivo

Desenvolver uma aplicação que simula as principais funcionalidades do Discord, com foco em comunicação em tempo real e organização por servidores e canais.

---

## 🚀 Funcionalidades

* Cadastro de usuários
* Login no sistema
* Criação de servidores
* Criação de canais
* Envio de mensagens em tempo real

---

## 📊 Diagramas

* Diagrama de Casos de Uso
* Diagrama de Sequência
* Diagrama de Estados

---

## 📱 Telas

* Tela de Login
* Tela de Registro
* Dashboard (Servidores)
* Tela de Chat
